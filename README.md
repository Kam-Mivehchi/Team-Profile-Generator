# Team Profile Generator

##Introduction
CLI team profile generator using Inquirer to gather user input. Designed to allow team leaders to quickly launch team websites.
Allows user to add 3 different types of employees: Managers, Engineers, Interns. Creates HTML dynamically with user entered data.

## Features
For Managers the user is prompted to enter a name, ID, office number and email.
For Engineers the user is prompted to enter a name, ID, github and email.
For Interns the user is prompted to enter a name, ID, school and email.
All email links open user-default mail application and starts a new message with the selected teammember as the recipient.
Github link opens in a new tab.
Input validation for all inquirer prompts.
